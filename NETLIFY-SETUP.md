# Подробная инструкция по настройке Netlify

## Шаг 1: Вход в Netlify

1. Откройте браузер и перейдите на **https://app.netlify.com**
2. Если у вас нет аккаунта:
   - Нажмите **"Sign up"** (Регистрация)
   - Выберите **"Sign up with GitHub"** (Рекомендуется!)
   - Разрешите доступ к вашему GitHub аккаунту
3. Если аккаунт уже есть - просто войдите

---

## Шаг 2: Создание нового сайта

1. На главной странице Netlify найдите большую кнопку **"Add new site"**
   - Она может быть вверху справа или в центре страницы
   - Или в левом меню найдите **"Sites"** → **"Add new site"**

2. Нажмите на **"Add new site"**

3. Появится меню с вариантами:
   - **"Import an existing project"** ← ВЫБЕРИТЕ ЭТО
   - "Deploy manually" (не выбирайте)

---

## Шаг 3: Подключение GitHub

1. После выбора "Import an existing project" вы увидите список провайдеров:
   - **GitHub** ← НАЖМИТЕ НА ЭТО
   - GitLab
   - Bitbucket
   - Azure DevOps

2. Если вы впервые подключаете GitHub:
   - Нажмите **"Authorize Netlify"** или **"Configure the Netlify GitHub App"**
   - Вас перенаправит на GitHub
   - Нажмите **"Authorize netlify"** (зеленая кнопка)
   - Разрешите доступ к репозиториям
   - Вас вернет обратно в Netlify

3. Если GitHub уже подключен:
   - Просто выберите **"GitHub"** из списка

---

## Шаг 4: Выбор репозитория

1. После авторизации вы увидите список ваших GitHub репозиториев

2. Найдите репозиторий **`yugdomstroi.ru`** или **`kbkhn0009-prog/yugdomstroi.ru`**

3. Нажмите на него

---

## Шаг 5: Настройка деплоя

После выбора репозитория откроется страница с настройками:

### Branch to deploy (Ветка для деплоя):
- Оставьте **`main`** (по умолчанию)
- Это ветка, из которой будет браться код

### Build command (Команда сборки):
- **ОСТАВЬТЕ ПУСТЫМ!** 
- Ваш сайт статический, сборка не нужна
- Просто удалите все, что там написано

### Publish directory (Директория публикации):
- Введите: **`.`** (одну точку)
- Это означает "корневая папка проекта"
- Или можно оставить пустым, если там уже стоит точка

### Advanced build settings (Дополнительные настройки):
- Нажмите **"Show advanced"** если хотите
- Но для вашего сайта это не нужно

---

## Шаг 6: Запуск деплоя

1. Проверьте настройки еще раз:
   - ✅ Branch: `main`
   - ✅ Build command: (пусто)
   - ✅ Publish directory: `.`

2. Нажмите большую зеленую кнопку **"Deploy site"**

3. Начнется процесс деплоя (займет 1-2 минуты)

---

## Шаг 7: Ожидание деплоя

1. Вы увидите страницу с прогрессом:
   - "Building your site..."
   - "Deploying your site..."

2. Дождитесь сообщения **"Site is live"** или **"Published"**

3. Netlify автоматически создаст адрес типа:
   - `random-name-12345.netlify.app`
   - Этот адрес будет показан на странице

---

## Шаг 8: Проверка сайта

1. Нажмите на адрес сайта (например, `random-name-12345.netlify.app`)

2. Откроется ваш сайт в новой вкладке

3. Проверьте, что все работает:
   - Главная страница открывается
   - Навигация работает
   - Изображения загружаются
   - Все страницы доступны

---

## Шаг 9: Подключение домена (опционально)

Если у вас есть домен (например, `yugdomstroi.ru`):

1. В панели Netlify выберите ваш сайт

2. Перейдите в **"Domain settings"** (в меню слева)

3. Нажмите **"Add custom domain"**

4. Введите ваш домен (например, `yugdomstroi.ru`)

5. Netlify покажет DNS записи, которые нужно добавить:
   - **A запись:** 
     - Name: `@` или оставьте пустым
     - Value: IP адрес (будет показан, например `75.2.60.5`)
   - **CNAME запись:**
     - Name: `www`
     - Value: `ваш-сайт.netlify.app` (адрес, который дал Netlify)

6. Добавьте эти записи у вашего регистратора домена:
   - Зайдите в панель управления доменом
   - Найдите раздел "DNS" или "Управление DNS"
   - Добавьте записи как указал Netlify

7. Подождите 5-60 минут для распространения DNS

8. Netlify автоматически выдаст SSL сертификат (HTTPS)

## Автоматический деплой

Теперь каждый раз, когда вы делаете изменения:

1. Внесите изменения в код локально

2. Выполните:
   ```bash
   git add .
   git commit -m "Описание изменений"
   git push
   ```

3. Netlify автоматически обнаружит изменения и задеплоит сайт за 1-2 минуты!

4. Вы увидите новый деплой в панели Netlify в разделе **"Deploys"**

## Решение проблем

### Сайт не деплоится
- Проверьте логи в разделе "Deploys" → выберите деплой → "Deploy log"
- Убедитесь, что Build command пустой
- Проверьте, что Publish directory = `.`

### Ошибка "Build failed"
- Убедитесь, что Build command пустой
- Проверьте, что все файлы закоммичены в Git

### Домен не подключается
- Проверьте DNS записи (может занять до 48 часов)
- Убедитесь, что домен не используется на другом хостинге
- Проверьте, что записи добавлены правильно

### Изменения не появляются после push
- Подождите 1-2 минуты
- Проверьте раздел "Deploys" - должен быть новый деплой
- Очистите кэш браузера (Ctrl+F5)
